## Numbers in Python
   
   ### In Python 2 main types of numbers:
      - Integer
      - Floating Point

   ### To know the type:
       type(N)

   Examples:
   - type(9)
   - type (0.9)

   ### Adding integers and floats gives the floats
       1 + 1.0 = 2.0

   ### Division always returns floats
       15/3 = 5.0

   ### Math operators
   | ***Symbol*** | ***Name*** |
   | :---         | :--- |
   | + | Add |
   | - | Subtract |
   | * | Multiply |
   | / | Divide |
   | ** | Exponential |
   | % | Modulo |
   | // | Integer Division |
