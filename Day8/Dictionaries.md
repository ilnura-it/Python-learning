# Dictionaties

   A data structure that consists of key value pairs

      cat = {

### Accessing Individual Values

      instructor = {
         "name": "Instructor",
         "own_dog": True,
         "num_courses": 4,
         "favourite_languages": "Python"
      }

      instructor["name"] # "Instructor"
      instructor["thing"] # KeyError
